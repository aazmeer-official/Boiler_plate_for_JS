# Boiler Plate for JS

Starter boilerplate for **Node.js + Express.js** projects with **EJS templates**, **MySQL connection**, **Faker.js**, and **UUID**. Quickly start projects without rewriting setup code.

**Repo:** [https://github.com/aazmeer-official/Boiler_plate_for_JS.git](https://github.com/aazmeer-official/Boiler_plate_for_JS.git)

---

## Features
- Express.js setup  
- EJS templating engine  
- Views folder connection  
- Public folder static setup  
- Method override support  
- MySQL database connection  
- Faker.js for dummy data  
- UUID for unique IDs  

---

## Installation
```bash
git clone https://github.com/aazmeer-official/Boiler_plate_for_JS.git
cd Boiler_plate_for_JS
npm install express ejs mysql method-override @faker-js/faker uuid
